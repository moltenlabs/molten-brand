## [0.1.0] - 2024-12-14

### Added

- Initial release
- Core color types (`Rgb`, `Rgba`, `Color`)
- Forge brand colors (`colors::forge`)
- Molten orange scale (`colors::molten`)
- Neutral gray scale (`colors::neutral`)
- Surface colors (`colors::surface`)
- Text colors (`colors::text`)
- Glass effect colors (`colors::glass`)
- Product tokens for Lair (`products::lair`)
- Product tokens for Hearth (`products::hearth`)
- Product tokens for Alloy (`products::alloy`)
- Semantic colors (success, warning, error, info)
- Agent status colors (`semantic::agent`)
- Typography tokens (families, sizes, weights, presets)
- Spacing scale and semantic aliases
- Optional `serde` feature for serialization
